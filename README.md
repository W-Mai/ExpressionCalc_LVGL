﻿## LVGL Expression Calculator

基于LVGL的表达式计算器

- 支持+ - * / \ % ^等双目运算
- 支持()
- 支持sqrt abs sin cos tan asin acos atan ln log log2 floor ceil sign等函数
- 支持常量 PI E
